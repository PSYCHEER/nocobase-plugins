# Comments Plugin

Advanced commenting system with GitHub-style changelog tracking.

## Features

- **Comments on any record** - Add comments to any database record
- **Threaded replies** - Nested comment threads
- **@Mentions** - Mention users with @username
- **Rich text editor** - Formatted text with attachments
- **Notifications** - Alert users about new comments and mentions
- **Permissions** - Control who can comment, edit, delete
- **GitHub-style changelog** - Track record changes with visual timeline
- **Activity feed** - See all activities and comments in one place

## Usage

1. Enable the plugin in your NocoBase installation
2. Add the Comments block to any page
3. Configure permissions for commenting
4. Start collaborating!
